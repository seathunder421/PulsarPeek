# PulsarPeek 🌀

**A Desktop Tool for Pulsar Data Browsing & Visualization**

Built on the ATNF Pulsar Catalogue, PulsarPeek enables quick exploration of pulsar parameters, distributions, and associations across different categories. It supports category filtering, single-source detail viewing, P-Ṗ diagram, interactive Sky Map, and CSV export — ideal for data retrieval and preliminary analysis. Runs as a native desktop application without a browser.

## Features

- 📋 **Category Filtering** — Filter pulsars by category and compare parameter characteristics across types
- 🔍 **Single-Source Detail Viewer** — Click any pulsar to view full parameter details (period, DM, magnetic field strength, etc.)
- 📊 **P-Ṗ Diagram & Sky Map** — Interactive P-Ṗ diagram and celestial sphere map
- 💾 **CSV Export** — One-click export of filtered results for citations and further analysis
- ⚡ **Native Desktop App** — Runs locally on macOS, no browser needed, data cached offline

## Requirements

- macOS 11.0+ (Apple Silicon / Intel)

## Install

1. Download \`PulsarPeek_v1.dmg\` from the repo root
2. Open the DMG and drag to Applications
3. Right-click → Open to bypass Gatekeeper on first launch
