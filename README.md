# PulsarPeek 🌀

**面向脉冲星数据的桌面浏览与可视化工具**

基于 ATNF Pulsar Catalogue 数据，方便快速查看不同类别脉冲星的参数、分布和关联信息。支持分类筛选、单源详情查看、P-Ṗ 图、Sky Map 以及 CSV 导出，适合做数据检索和初步分析。采用本地桌面窗口运行，打开后不依赖浏览器，适合在 Mac 上直接使用。

## Features

- 📋 **分类筛选** — 按类别快速筛选脉冲星，对比不同类型参数特征
- 🔍 **单源详情查看** — 点击任意脉冲星查看完整参数详情（周期、色散量、磁场强度等）
- 📊 **P-Ṗ 图 & Sky Map** — 交互式 P-Ṗ 图和天球分布图
- 💾 **CSV 导出** — 筛选结果一键导出，方便论文引用和进一步分析
- ⚡ **本地桌面运行** — 原生 macOS 应用，不依赖浏览器，数据本地缓存

## Requirements

- macOS 11.0+ (Apple Silicon / Intel)

## Install

1. Download `PulsarPeek_v1.dmg` from repo root
2. Open the DMG and drag to Applications
3. Right-click → Open to bypass Gatekeeper on first launch
