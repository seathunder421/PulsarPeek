# PulsarCatalogue 🌀

**脉冲星数据目录应用 — 快速浏览、搜索和分析脉冲星数据**

## 简介

PulsarCatalogue 是一个专为脉冲星研究设计的 macOS 应用，提供直观的界面来浏览、搜索和可视化脉冲星数据目录。支持高效的本地数据管理与实时搜索，适用于天体物理研究中的脉冲星数据筛选与分析工作。

## 功能

- 📋 **脉冲星目录浏览** — 快速加载和浏览脉冲星数据
- 🔍 **多维度搜索** — 按名称、周期、色散量等参数筛选
- 📊 **数据可视化** — 关键参数分布图表
- 💾 **本地存储** — 数据缓存，离线可用
- ⚡ **高性能** — 针对大规模数据优化

## 系统要求

- macOS 11.0 (Big Sur) 或更高版本
- Apple Silicon 或 Intel 处理器

## 安装

1. 下载最新版本的 `PulsarCatalogue.dmg`
2. 打开 DMG 文件
3. 将应用拖入 Applications 文件夹
4. 首次打开可能需要右键 → 打开以绕过 Gatekeeper
