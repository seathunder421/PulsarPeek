# PulsarPeek 🌀

**A Desktop Tool for Pulsar Data Browsing & Visualization**

Built on the [ATNF Pulsar Catalogue](https://www.atnf.csiro.au/research/pulsar/psrcat/), PulsarPeek enables quick exploration of pulsar parameters, distributions, and associations across different categories.

Runs as a native macOS desktop application — no browser needed, data cached locally for offline use.

---

## ✨ Features

| Feature | Description |
|---|---|
| 📋 **Category Filtering** | Filter pulsars by category and compare parameter characteristics across types |
| 🔍 **Single-Source Detail Viewer** | Click any pulsar to view full parameters — period, DM, magnetic field, and more |
| 📊 **P-Ṗ Diagram** | Interactive period–period-derivative diagram for pulsar population analysis |
| 🌌 **Sky Map** | Interactive celestial sphere map showing spatial distribution |
| 💾 **CSV Export** | One-click export of filtered results for citations and further analysis |
| ⚡ **Native macOS App** | Runs locally, no browser, data cached offline |

---

## 📦 Requirements

- macOS 11.0+ (Big Sur or later)
- Apple Silicon (M1+) **or** Intel

## ⬇️ Download

[**Download PulsarPeek_v2.dmg**](PulsarPeek_v2.dmg) — ~16 MB

### 📦 Install

1. Download `PulsarPeek_v2.dmg`
2. 双击打开 DMG 文件
3. 将 `PulsarPeek.app` 拖入 Applications 文件夹
4. 首次打开会弹出 macOS 安全提示：

> **"PulsarPeek" was blocked to protect your Mac**

**解决方法**：右键（或按住 Ctrl 点击）`PulsarPeek.app` → 选择 **打开** → 在弹出的对话框里点击 **打开** 即可。

或者：系统设置 → 隐私与安全性 → 找到 "PulsarPeek" 被阻止的提示 → 点击 **仍要打开**。

只需操作一次，以后正常双击打开即可。
