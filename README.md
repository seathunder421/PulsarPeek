# PulsarPeek 🌀

**A macOS app for browsing, searching, and analyzing pulsar catalog data.**

## Features

- 📋 **Catalog browser** — fast loading and navigation
- 🔍 **Multi-dimensional search** — filter by name, period, DM, etc.
- 📊 **Data visualization** — key parameter distribution charts
- 💾 **Local caching** — offline-ready
- ⚡ **Performance optimized** — handles large datasets

## Requirements

- macOS 11.0+ (Apple Silicon / Intel)

## Install

1. Download `PulsarCatalogue_v1.dmg` from repo root
2. Open the DMG and drag to Applications
3. Right-click → Open to bypass Gatekeeper on first launch
