PulsarPeek 🌀

A Desktop Tool for Pulsar Data Browsing & Visualization

Built on the ATNF Pulsar Catalogue — explore pulsar parameters, distributions, and associations across categories, right from your desktop.

📦 Install

1

Double-click the downloaded PulsarPeek_v2.dmg

2

Drag PulsarPeek.app into the Applications folder

3

First launch will trigger a macOS security warning — see fix below

⚠️ "PulsarPeek" was blocked to protect your Mac

Since PulsarPeek is not Apple-notarized, macOS blocks the first launch. This is expected. Two ways to fix it:

Option 1 (recommended): In Finder, right-click (or Ctrl+click) PulsarPeek.app → select Open → click Open in the confirmation dialog. It will run normally from then on.

Option 2: System Settings → Privacy & Security → scroll down to the Security section → find "PulsarPeek was blocked" → click Open Anyway → enter your password.

✅ Only needed once — subsequent launches work with a normal double-click

Features

Category Filtering

Quickly filter pulsars by category and compare parameter characteristics across different types — MSP, young, binary, and more.

Single-Source Detail Viewer

Click any pulsar to view full parameters — period, dispersion measure, characteristic age, magnetic field strength, and more.

P–Ṗ Diagram & Sky Map

Interactive period–period-derivative diagram and celestial sphere map for pulsar population analysis and spatial distribution.

CSV Export

One-click CSV export of filtered results for citations, publication figures, and further analysis in your favorite tools.

Native Desktop App

Runs natively on macOS. No browser required. Data cached locally for full offline use.
